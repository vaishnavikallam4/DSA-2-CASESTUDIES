# Flipkart Product Search and Inventory Management System using AVL Tree

## Student Details
- Roll No: 2520030198
- Subject: DSA-2
- CO: CO-1

## Problem Statement
Flipkart stores millions of products and requires fast product search and inventory updates. AVL Tree is used to maintain balanced indexing and efficient searching.

## Features
- AVL Tree Implementation
- Product Search
- Product Insertion
- Inorder Traversal
- AVL Rotations (LL, RR, LR, RL)

## Output
Product IDs sorted successfully.
Search operation completed.

## Time Complexity
- Search: O(log n)
- Insert: O(log n)
- Delete: O(log n)
